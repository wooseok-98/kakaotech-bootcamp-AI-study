# 한 줄 정리

| 주제 | 한 줄 정리 |
| --- | --- |
| FastAPI | Python으로 비동기 API 서버를 빠르게 만들 수 있는 웹 프레임워크 |
| pyproject.toml | Python 프로젝트의 의존성과 설정을 관리하는 표준 설정 파일 |
| Gunicorn | 운영 환경에서 여러 워커 프로세스를 관리하는 Python 웹 서버 |
| HTTP Message | 클라이언트와 서버가 주고받는 요청/응답 데이터의 형식 |
| HTTP Request Method | 서버에 어떤 작업을 요청할지 나타내는 동사 (GET, POST, PUT, DELETE 등) |
| HTTP Status Code | 서버가 요청을 처리한 결과를 숫자로 나타내는 코드 (200, 404, 500 등) |
| HTTP URL | 서버의 특정 리소스를 가리키는 주소 |
| Query String | URL 뒤에 `?key=value` 형태로 붙이는 검색·필터용 파라미터 |
| Path Variable | URL 경로 안에 값을 넣어 리소스를 특정하는 방식 (`/users/1`) |
| JSON | 데이터를 키-값 쌍으로 표현하는 텍스트 기반 데이터 교환 형식 |
| REST API | HTTP 규칙을 따라 자원을 URL로 표현하고 Method로 조작하는 API 설계 방식 |
