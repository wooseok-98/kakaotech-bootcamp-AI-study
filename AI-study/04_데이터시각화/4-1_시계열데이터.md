## 시계열 데이터 (Time Series Data)

시간의 흐름에 따라 일정한 간격(초, 분, 시간, 일, 월, 연 등)으로 측정된 연속적인 데이터

> 시간을 독립 변수(X축), 관측값을 종속 변수(Y축)로 설정해 분석

### 대표적인 예시

| 예시 | 시간 축 | 관측 값 |
|------|---------|---------|
| 주식 가격 변화 | 일/분/초 단위 | 종가, 시가, 고가, 저가 |
| 날씨 (기온, 강수량) | 일/시간 단위 | 기온(℃), 강수량(mm), 습도(%) |
| 웹사이트 트래픽 | 시간 단위 | 접속자 수, 페이지뷰 |
| IoT 센서 데이터 | 초/분 단위 | 온도, 진동, 소음, 습도 |
| 환자 생체 데이터 | 분 단위 | 심박수, 혈압, 체온 |

### 사용 이유

| 이유 | 설명 |
|------|------|
| 시간에 따른 변화 분석 | 주식 변동성, 기온 변화, 트래픽 패턴 파악 |
| 추세(Trend) 식별 | 장기적인 증가·감소 패턴 파악 |
| 계절성(Seasonality) 분석 | 특정 시점마다 반복되는 패턴 탐지 |
| 이상치(Outlier) 탐지 | 급격한 변화(급등·급락) 감지 |
| 이벤트 감지 | 특정 사건이 데이터에 미치는 영향 분석 |

### 기본 시각화

```python
import seaborn as sns
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)
date_range = pd.date_range(start="2023-01-01", periods=100, freq="D")
data = np.cumsum(np.random.randn(100))  # 누적합으로 시계열 형태 시뮬레이션

df = pd.DataFrame({"Date": date_range, "Value": data})

plt.figure(figsize=(10, 5))
sns.lineplot(x="Date", y="Value", data=df, color="blue", marker="o")
plt.xlabel("Date")
plt.ylabel("Value")
plt.title("Time Series Line Plot")
plt.xticks(rotation=45)
plt.show()
```

> `np.cumsum()` — 배열의 누적합을 계산하는 함수. 주식 가격처럼 이전 값에 다음 값이 더해지는 데이터를 시뮬레이션할 때 사용

### 이동 평균 시각화

```python
df["Moving_Avg"] = df["Value"].rolling(window=7).mean()

plt.figure(figsize=(10, 5))
sns.lineplot(x="Date", y="Value", data=df, label="Original Data", color="gray")
sns.lineplot(x="Date", y="Moving_Avg", data=df, label="7-Day Moving Average", color="red")
plt.title("Time Series with Moving Average")
plt.legend()
plt.xticks(rotation=45)
plt.show()
```

### 이상치 탐지 (IQR 활용)

**IQR(Interquartile Range)** — Q3(75%)와 Q1(25%)의 차이로 이상치 경계를 정의

```python
Q1 = df["Value"].quantile(0.25)
Q3 = df["Value"].quantile(0.75)
IQR = Q3 - Q1
k = 1.5  # 일반적인 Tukey 기준값 (k < 1.5이면 더 민감하게 탐지)

lower_bound = Q1 - k * IQR
upper_bound = Q3 + k * IQR

df["Outlier"] = (df["Value"] < lower_bound) | (df["Value"] > upper_bound)
outliers = df[df["Outlier"]]

plt.figure(figsize=(10, 5))
sns.lineplot(x="Date", y="Value", data=df, label="Original Data", color="blue")
sns.scatterplot(x="Date", y="Value", data=outliers, color="red", label="Outliers", s=100)
plt.title("Time Series with Outlier Detection")
plt.legend()
plt.xticks(rotation=45)
plt.show()
```

## 리샘플링 (Resampling)

시계열 데이터의 분석을 위해 데이터의 시간 간격을 재조정하여 다운샘플링(축소)하거나 업샘플링(확대)하는 과정

| 개념 | 설명 |
|------|------|
| **다운샘플링(Downsampling)** | 더 긴 시간 간격으로 집계 (초 → 분, 일 → 주) |
| **업샘플링(Upsampling)** | 더 짧은 시간 간격으로 확장 (일 → 시간) |
| **집계(Aggregation)** | 다운샘플링 시 평균·합계·최대값 등으로 요약 |
| **보간(Interpolation)** | 업샘플링 시 생성된 결측값(NaN) 채우기 |

```
초 ↕ 분 ↕ 시 ↕ 일 ↕ 주 ↕ 월
← 업샘플링 | 다운샘플링 →
```

### 기본 문법

```python
df.resample(시간_간격).집계함수()
```

| 시간 간격 | 의미 |
|---------|------|
| `"s"` | 초 |
| `"min"` | 분 |
| `"h"` | 시간 |
| `"D"` | 일 |
| `"W"` | 주 |
| `"ME"` | 월말 |

### 사용 방법

```python
import pandas as pd
import numpy as np

date_rng = pd.date_range(start="2024-01-01", end="2024-01-10", freq="3h")
df = pd.DataFrame({"datetime": date_rng, "value": np.random.randint(10, 100, size=len(date_rng))})
df.set_index("datetime", inplace=True)

# 다운샘플링 — 하루 단위 평균
df_daily = df.resample("D").mean()

# 업샘플링 — 1시간 단위 확장 (빈 값은 NaN)
df_hourly = df.resample("h").asfreq()

# 선형 보간으로 NaN 채우기
df_hourly_interp = df_hourly.interpolate(method="linear")
```

### 보간법 종류

| 보간법 | 사용법 | 설명 |
|--------|--------|------|
| 선형 | `interpolate(method="linear")` | 두 점을 직선으로 연결해 중간값 추정 |
| 전방 채우기 | `fillna(method="ffill")` | 이전 값으로 NaN 채움 |
| 후방 채우기 | `fillna(method="bfill")` | 다음 값으로 NaN 채움 |
| 다항식 | `interpolate(method="polynomial", order=n)` | 고차 다항식 근사 (과적합 주의) |
| 스플라인 | `interpolate(method="spline", order=n)` | 구간별 3차 다항식으로 부드럽게 연결 |
| 시간 | `interpolate(method="time")` | 시간 간격을 고려해 보간 |

> **다항식 vs 스플라인**: 다항식은 전체를 하나의 고차 함수로 근사해 데이터가 많아지면 진동(런게 현상)이 생김. 스플라인은 구간별 낮은 차수 다항식을 이어 붙여 안정적

> 선형 보간은 기존 직선 위에 점을 추가하는 방식이므로, 원본과 그래프가 겹쳐 보이는 것이 정상

## 이동평균 (Moving Average)

시계열 데이터의 변동성을 완화하고 추세를 분석하기 위해 일정한 구간(Window)의 평균값을 계산하는 기법

> **윈도우(Window)** — 평균을 계산할 연속된 데이터 개수. `window=7`이면 최근 7개 데이터를 기준으로 이동하며 계산

| 유형 | 계산 방식 | 특징 |
|------|---------|------|
| **SMA** (단순 이동평균) | 동일한 가중치로 평균 | 계산이 단순, 최근 변화 반영이 느림 |
| **EMA** (지수 이동평균) | 최근 값에 지수적으로 높은 가중치 | 변동성을 빠르게 반영 |
| **WMA** (가중 이동평균) | 최근 값일수록 선형적으로 높은 가중치 | SMA와 EMA의 중간 반응 속도 |

**반응 속도**: EMA > WMA > SMA

### 사용 방법

```python
import pandas as pd
import numpy as np

date_rng = pd.date_range(start="2024-01-01", end="2024-01-20", freq="D")
df = pd.DataFrame({"datetime": date_rng, "value": np.random.randint(50, 150, size=len(date_rng))})
df.set_index("datetime", inplace=True)

# 단순 이동평균 (SMA) — 처음 6개는 NaN
df["SMA_7"] = df["value"].rolling(window=7).mean()

# 지수 이동평균 (EMA) — α = 2 / (span + 1), 초기값부터 바로 계산
df["EMA_7"] = df["value"].ewm(span=7, adjust=False).mean()

# 가중 이동평균 (WMA)
weights = np.arange(1, 8)  # [1, 2, 3, 4, 5, 6, 7]
df["WMA_7"] = df["value"].rolling(window=7).apply(
    lambda x: np.dot(x, weights) / weights.sum(), raw=True
)
```

EMA 계산 예시 (α = 0.25, span=7):

| 날짜 | 실제값 | 계산식 | EMA 결과 |
|------|--------|--------|---------|
| 1일차 | 128 | 초기값 | 128.00 |
| 2일차 | 148 | 0.25×148 + 0.75×128 | 133.00 |
| 3일차 | 104 | 0.25×104 + 0.75×133 | 125.75 |

### 비교 시각화

```python
import matplotlib.pyplot as plt

plt.figure(figsize=(12, 6))
plt.plot(df.index, df["value"], marker="o", linestyle="-", label="Original Data", alpha=0.5)
plt.plot(df.index, df["SMA_7"], linestyle="--", label="SMA (7-day)")
plt.plot(df.index, df["EMA_7"], linestyle=":", label="EMA (7-day)", color="red")
plt.plot(df.index, df["WMA_7"], linestyle="-.", label="WMA (7-day)", color="green")
plt.title("Comparison of Moving Averages")
plt.legend()
plt.grid()
plt.xticks(rotation=45)
plt.show()
```

## 금융 데이터 (Financial Data)

주식 가격, 거래량, 환율, 금리 등 금융 시장에서 발생하는 시계열 기반의 데이터

### 주요 구성 요소

| 구성 요소 | 설명 |
|---------|------|
| **주식 가격(Stock Prices)** | 시가(Open), 고가(High), 저가(Low), 종가(Close), 조정 종가(Adjusted Close) |
| **거래량(Trading Volume)** | 일정 기간 동안 거래된 주식의 총 수량 |
| **지수(Market Index)** | S&P 500, NASDAQ, KOSPI 등 시장 전체 흐름을 나타내는 가중 평균 |
| **환율(Forex)** | 두 국가 통화 간 교환 비율 |
| **금리(Interest Rate)** | 중앙은행 기준 금리 및 채권 시장 금리 |
| **파생상품(Derivatives)** | 선물(Futures), 옵션(Options), 스왑(Swaps) 등 |
| **경제 지표** | GDP 성장률, 실업률, CPI(소비자물가지수) 등 |

### 수집 주기에 따른 데이터 유형

| 유형 | 설명 |
|------|------|
| **틱(Tick) 데이터** | 초 단위 이하의 모든 거래 체결 정보 |
| **분봉(Minute Bar)** | 분 단위로 시가·고가·저가·종가·거래량 집계 |
| **일봉(Daily Bar)** | 하루 단위 집계 — 가장 일반적인 형태 |
| **주봉(Weekly Bar)** | 한 주 단위 집계 |
| **월봉(Monthly Bar)** | 한 달 단위 집계 |

### 샘플 데이터 생성 (틱 → 분봉 → 일봉 → 주봉 → 월봉)

```python
import pandas as pd
import numpy as np

# 틱 데이터 생성 (1년, 초 단위)
date_rng = pd.date_range(start="2023-01-02", end="2023-12-31 23:59:59", freq="s")

np.random.seed(42)
open_prices = np.random.uniform(100, 200, size=len(date_rng))
high_prices = open_prices + np.random.uniform(0, 5, size=len(date_rng))
low_prices = open_prices - np.random.uniform(0, 5, size=len(date_rng))
close_prices = (open_prices + high_prices + low_prices) / 3 + np.random.uniform(-2, 2, size=len(date_rng))
volumes = np.random.randint(100, 1000, size=len(date_rng))

df_tick = pd.DataFrame({
    "DateTime": date_rng,
    "Open": open_prices, "High": high_prices,
    "Low": low_prices, "Close": close_prices, "Volume": volumes
})
df_tick.set_index("DateTime", inplace=True)
# 데이터 크기가 크므로 실행 시간이 오래 걸리는 것이 정상

# 틱 → 분봉
df_minute = df_tick.resample("min").agg({
    "Open": "first", "High": "max", "Low": "min", "Close": "last", "Volume": "sum"
})

# 분봉 → 5분봉
df_5min = df_minute.resample("5min").agg({
    "Open": "first", "High": "max", "Low": "min", "Close": "last", "Volume": "sum"
})

# 분봉 → 일봉 + 이동평균 + 변동성
df_daily = df_minute.resample("D").agg({
    "Open": "first", "High": "max", "Low": "min", "Close": "last", "Volume": "sum"
})
df_daily["SMA_30"] = df_daily["Close"].rolling(window=30).mean()
df_daily["Volatility"] = df_daily["Close"].rolling(window=30).std()

# 일봉 → 주봉
df_weekly = df_daily.resample("W").agg({
    "Open": "first", "High": "max", "Low": "min", "Close": "last", "Volume": "sum"
})
df_weekly["SMA_20"] = df_weekly["Close"].rolling(window=20).mean()
df_weekly["EMA_20"] = df_weekly["Close"].ewm(span=20, adjust=False).mean()
df_weekly["Volatility"] = df_weekly["Close"].rolling(window=20).std()

# 일봉 → 월봉
df_monthly = df_daily.resample("ME").agg({
    "Open": "first", "High": "max", "Low": "min", "Close": "last", "Volume": "sum"
})
df_monthly["SMA_12"] = df_monthly["Close"].rolling(window=12).mean()
df_monthly["EMA_12"] = df_monthly["Close"].ewm(span=12, adjust=False).mean()
df_monthly["Volatility"] = df_monthly["Close"].rolling(window=12).std()
```

### 캔들 차트 시각화 (mplfinance)

**캔들 차트(Candlestick Chart)** — 시가·고가·저가·종가를 몸통(Body)과 꼬리(Wick)로 표현하는 금융 전용 차트. 상승(양봉)은 초록색, 하락(음봉)은 빨간색으로 표시

**mplfinance** — matplotlib 기반의 금융 데이터 시각화 라이브러리. 캔들 차트, 거래량, 이동평균 등을 간결하게 시각화


```python
df_daily_filtered = df_daily.iloc[-15:]

ap_daily = [
    mpf.make_addplot(df_daily_filtered["SMA_30"], panel=0, label="30-Day SMA", color="black", width=1.5),
```
