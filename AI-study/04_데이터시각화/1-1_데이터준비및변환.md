## 데이터 시각화 (Data Visualization)

데이터를 그래프나 차트와 같은 시각적 형식으로 표현하여 패턴, 추세, 인사이트를 효과적으로 전달하는 과정

### 유형

| 유형 | 도구 |
|------|------|
| 양적 데이터 | 선 그래프, 막대 그래프, 히스토그램 |
| 질적 데이터 | 파이 차트, 트리맵 |
| 시계열 데이터 | 선 그래프, 영역 그래프 |
| 지리적 데이터 | 지도 |

### 사용 이유

- 패턴·추세 파악 (이상치, 계절성 등)
- 그룹 간 비교 및 분류 용이
- 박스플롯 등으로 데이터 품질 문제 조기 발견

## 정형 데이터 (Structured Data)

표나 데이터베이스처럼 **행(Row)과 열(Column)의 구조**를 가진 체계적인 데이터

- 스키마(Schema) 기반 — 열 이름, 데이터 타입, 제약 조건이 미리 정의됨
- CSV, 엑셀, RDBMS(MySQL, SQLite 등)에 저장

### Pandas 활용 흐름

```
불러오기 → 탐색 → 필터링·변환 → 정리·가공 → 정렬·그룹화 → 저장
```

```python
import pandas as pd

# 불러오기
df = pd.read_csv('data.csv')

# 탐색
df.info()
df.head()
df.describe()

# 필터링 및 변환
df_filtered = df[df['나이'] >= 30]
df['연봉_원'] = df['연봉'] * 10000

# 정리 및 가공
df.fillna('없음', inplace=True)
df.drop_duplicates(inplace=True)
df['가입일'] = pd.to_datetime(df['가입일'])

# 정렬 및 그룹화
df.sort_values(by='연봉', ascending=False)
df.groupby('부서')['연봉'].mean()

# 저장
df.to_csv('processed_data.csv', index=False)
```

## 비정형 데이터 (Unstructured Data)

고정된 구조 없이 **텍스트, 이미지, 영상, 오디오** 등 다양한 형식으로 존재하는 데이터

- 정형 데이터만으로 포착할 수 없는 패턴과 의미를 해석하기 위해 사용
- 분석이 복잡해 NLP, 컴퓨터 비전 등 별도 기술 필요

> **반정형 데이터**: 정형과 비정형의 중간. JSON, XML, YAML 등 — 스키마는 없지만 특정 규칙을 따름

### 처리 흐름

```
수집 → 전처리 → 변환(정형화) → 분석 → 저장
```

```python
import requests, re, pandas as pd
from collections import Counter

# 수집 — API
data = requests.get("https://jsonplaceholder.typicode.com/posts").json()
df = pd.DataFrame(data)

# 분석 — 텍스트 길이
df['제목_길이'] = df['title'].apply(len)
df['본문_길이'] = df['body'].apply(len)

# 텍스트 정제
clean = re.sub(r"[^a-zA-Z0-9\s]", "", df['title'][0])

# 단어 빈도 분석
word_counts = Counter(clean.split())
print(word_counts.most_common(10))

# 저장
df.to_csv("result.csv", index=False, encoding='utf-8')
```
