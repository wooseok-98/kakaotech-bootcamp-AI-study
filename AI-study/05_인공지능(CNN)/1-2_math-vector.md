## N차 함수

**최고차항의 차수가 N인 다항 함수**

| 용어 | 의미 |
| --- | --- |
| 차 (次) | 함수의 최고차항의 차수 |
| 함수 | 입력값에 따라 출력값을 만들어내는 이항 관계 |
| N차 함수 | 최고차항의 차수가 N인 다항 함수 |

### 차수별 형태

| 차수 | 형태 | 그래프 |
| --- | --- | --- |
| 1차 | `y = ax + b` | 직선 |
| 2차 | `y = ax² + bx + c` | 포물선 (한 번의 곡선) |
| 3차 | `y = ax³ + bx² + cx + d` | 두 번의 곡선 |

> 선형(직선) vs 비선형(곡선): 1차 함수는 선형, 2차 이상은 비선형

### 딥러닝에서의 역할

| 이유 | 설명 |
| --- | --- |
| 복잡한 패턴 학습 | N차 함수는 데이터의 복잡한 비선형 패턴을 학습하는 데 중요한 역할을 함 |
| 모델의 유연성 증가 | 다양한 형태의 데이터에 대해 높은 적합성을 가지게 함 |
| 오버피팅·언더피팅 방지 | 적절한 차수 선택으로 모델의 일반화 성능 향상 가능 |
| 활성화 함수의 기반 | ReLU, Sigmoid 등 비선형 활성화 함수도 N차 함수의 특성을 가짐 |

> 너무 낮은 차수 → 과소적합(underfitting) / 너무 높은 차수 → 과적합(overfitting)

### 코드 예시

```python
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-10, 10, 400)

y1 = x + 1             # 1차 함수
y2 = x**2 + x + 1     # 2차 함수 (**은 거듭제곱)
y3 = x**3 + x**2 + x + 1  # 3차 함수

plt.figure(figsize=(10, 6))

plt.subplot(3, 1, 1)
plt.plot(x, y1, label='y = x + 1')
plt.title('1차 함수')
plt.legend()

plt.subplot(3, 1, 2)
plt.plot(x, y2, label='y = x² + x + 1', color='orange')
plt.title('2차 함수')
plt.legend()

plt.subplot(3, 1, 3)
plt.plot(x, y3, label='y = x³ + x² + x + 1', color='green')
plt.title('3차 함수')
plt.legend()

plt.tight_layout()
plt.show()
```

| 함수 | 설명 |
| --- | --- |
| `np.linspace(start, stop, num)` | start ~ stop 사이를 num개의 균등 간격으로 생성 |
| `plt.subplot(rows, cols, index)` | 하나의 Figure를 여러 구역으로 나누어 각각 그래프를 그림 |
| `plt.tight_layout()` | 서브플롯 사이의 간격을 자동 조정하여 겹치지 않도록 함 |

## 벡터

**크기(얼마나 큰가)와 방향(어디로 향하는가)을 동시에 가지는 양**

머신러닝에서는 **입력 데이터를 수치적으로 표현하는 기본 단위**로 사용됨

| 데이터 종류 | 벡터 표현 |
| --- | --- |
| 이미지 | 각 픽셀 값이 벡터의 요소 |
| 텍스트 | 단어의 등장 빈도나 임베딩 값이 벡터의 요소 |
| 사람 정보 | [키, 몸무게, 나이] → `[175, 68, 29]` |

> 벡터는 고차원 공간 속의 하나의 점(point)으로 이해할 수 있음

### 사용 이유

| 이유 | 설명 |
| --- | --- |
| 수치적 표현 | 데이터를 수치화하여 수학적 연산과 분석을 용이하게 함 |
| 고차원 데이터 처리 | 다차원 데이터를 간편하게 표현하고 처리 가능 |
| 유사성·거리 분석 | 벡터 공간에서 데이터 간의 관계(유사성, 거리)를 분석하여 패턴 파악 가능 |
| 선형 대수 연산 | 행렬과의 연산을 통해 다양한 알고리즘 구현 가능 |

### 코드 예시

```python
import numpy as np

# 벡터 생성 (키, 몸무게, 나이)
person  = np.array([175, 68, 29])
another = np.array([180, 70, 25])

print("두 사람의 차이:", person - another)
# [-5 -2  4]

print("유클리드 거리:", np.linalg.norm(person - another))
# 6.708...
```

> **유클리드 거리**: 여러 특징(키, 몸무게, 나이)의 차이를 하나의 숫자로 합산한 두 벡터 간의 직선 거리
> 값이 작을수록 두 데이터가 유사하고, 값이 클수록 차이가 큼

## 벡터 · 행렬 · 텐서

| 용어 | 차원 | 설명 | 주요 사용처 |
| --- | --- | --- | --- |
| 벡터 | 1차원 | 크기와 방향을 갖는 1차원 배열 | 특징 벡터, 단어 임베딩, 픽셀 값 |
| 행렬 | 2차원 | 행과 열로 구성된 2차원 배열 | 이미지 데이터, 선형 방정식의 계수 |
| 텐서 | N차원 | 여러 차원을 포함하는 다차원 배열 | 딥러닝의 이미지·비디오·음성 데이터 처리 |

```python
import numpy as np

# 벡터 (1차원)
vector = np.array([1.5, 2.3, 3.0])

# 행렬 (2차원)
matrix = np.array([[1.5, 2.3, 3.0],
                   [4.1, 5.6, 6.2]])

# 텐서 (3차원)
tensor = np.array([[[1.5, 2.3, 3.0], [4.1, 5.6, 6.2]],
                   [[7.1, 8.4, 9.3], [10.5, 11.2, 12.7]]])

print(vector.shape)  # (3,)
print(matrix.shape)  # (2, 3)
print(tensor.shape)  # (2, 2, 3)
```
