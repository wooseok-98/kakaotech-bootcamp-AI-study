# 인공지능(CNN) 한 줄 정리

---

## 들어가며

| 개념 | 한 줄 정리 |
| --- | --- |
| TensorFlow | Google이 개발한 딥러닝 엔진으로, 모든 데이터를 텐서로 다루며 자동 미분과 GPU 가속을 지원 |
| Keras | TensorFlow 위에서 동작하는 고수준 API로, 층을 쌓는 방식으로 딥러닝 모델을 직관적으로 설계·학습 가능 |
| PyTorch | 동적 계산 그래프와 직관적인 코딩 스타일을 지원하는 딥러닝 라이브러리 — 연구·실험 환경에서 널리 사용 |
| Kaggle | 전 세계 데이터 과학자들이 데이터셋을 공유하고 분석 대회에 참가할 수 있는 플랫폼 |
| 수학적 함수 (N차 함수) | 최고차항의 차수가 N인 다항 함수로, 딥러닝에서 데이터의 복잡한 비선형 관계를 표현하는 데 사용 |
| 벡터 | 1차원 배열로 표현되는 데이터 구조 — 머신러닝에서 입력 데이터를 수치적으로 표현하는 기본 단위 |

## 데이터 전처리

| 개념 | 한 줄 정리 |
| --- | --- |
| Data Preprocessing | 분석·모델 학습을 위해 원본 데이터를 정제하고 변환하는 과정 (결측값 처리, 이상치 제거, 정규화, 원-핫 인코딩, 분할 포함) |
| Data Augmentation | 기존 데이터를 회전·뒤집기·색상 변형 등으로 변형하여 새로운 데이터를 생성하는 기법 — 데이터 부족 문제 해결 및 과적합 방지 |
| Dataset Split | 전체 데이터를 학습·검증·테스트 셋으로 나누어 모델의 일반화 성능을 객관적으로 평가하는 과정 |

## 머신러닝 기초

| 개념 | 한 줄 정리 |
| --- | --- |
| Machine Learning | 데이터의 패턴을 스스로 학습하여 예측·분류하는, 경험을 통해 자동으로 개선하는 컴퓨터 알고리즘의 연구 분야 |
| Random Forest | 배깅 방식으로 여러 의사결정 트리를 학습하고 다수결·평균으로 결합하여 과적합을 줄이고 일반화 성능을 높이는 앙상블 모델 |
| K-NN | 새로운 데이터와 가장 가까운 K개 이웃의 라벨을 기반으로 분류·예측하는, 학습 단계에서 파라미터를 학습하지 않는 메모리 기반 알고리즘 |
| SVM | 마진을 최대화하는 초평면을 찾아 데이터를 최적으로 분리하는 지도 학습 알고리즘 — 서포트 벡터만으로 결정 경계를 정의 |
| Naive Bayes | 모든 특성이 서로 독립이라고 가정하고 베이즈 정리로 각 클래스에 속할 확률을 계산하는 분류 모델 — 빠르고 텍스트 분류에 강함 |

## 딥러닝 기초

| 개념 | 한 줄 정리 |
| --- | --- |
| Deep Learning | 여러 층의 신경망을 통해 데이터의 비선형 패턴을 자동으로 학습하여 예측·분류하는 머신러닝의 하위 분야 |
| Perceptron | ANN의 기본 단위 — 입력값에 가중치를 곱한 가중합을 활성화 함수에 적용하여 단일 출력을 생성하는 선형 이진 분류기 |
| Activation Function | 뉴런의 가중합을 비선형 변환하여 신경망이 복잡한 패턴을 학습할 수 있게 하는 함수 (Sigmoid, Tanh, ReLU) |
| ANN | 뇌의 뉴런 구조를 모방한 알고리즘 — 입력층·은닉층·출력층을 거쳐 데이터의 패턴을 학습하고 예측을 수행 |
| FCNN | 각 층의 모든 뉴런이 다음 층의 모든 뉴런과 완전히 연결된 신경망 구조 — ANN의 한 종류 |
| FCL | ANN에서 모든 입력 뉴런이 모든 출력 뉴런과 연결된 계층 (TensorFlow: Dense Layer / PyTorch: Linear Layer) |
| Loss Function | 모델의 예측값과 실제값 간의 차이를 정량적으로 측정하는 함수 — 학습 과정에서 가중치 조정의 기준이 됨 |
| Backpropagation | 출력에서 발생한 손실을 체인 룰(Chain Rule)로 역방향 전파하여 각 가중치의 기울기(Gradient)를 계산하는 알고리즘 |
| Optimizer | 역전파로 계산된 기울기를 바탕으로 실제 가중치를 업데이트하여 손실을 최소화하는 최적화 알고리즘 |
