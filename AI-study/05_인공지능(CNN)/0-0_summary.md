| 벡터 | 1차원 배열로 표현되는 데이터 구조 — 머신러닝에서 입력 데이터를 수치적으로 표현하는 기본 단위 |

## 데이터 전처리

| 개념 | 한 줄 정리 |
| --- | --- |
| Data Preprocessing | 분석·모델 학습을 위해 원본 데이터를 정제하고 변환하는 과정 (결측값 처리, 이상치 제거, 정규화, 원-핫 인코딩, 분할 포함) |

| Dataset Split | 전체 데이터를 학습·검증·테스트 셋으로 나누어 모델의 일반화 성능을 객관적으로 평가하는 과정 |

## 머신러닝 기초

| 개념 | 한 줄 정리 |
| --- | --- |
| Machine Learning | 데이터의 패턴을 스스로 학습하여 예측·분류하는, 경험을 통해 자동으로 개선하는 컴퓨터 알고리즘의 연구 분야 |
| Random Forest | 배깅 방식으로 여러 의사결정 트리를 학습하고 다수결·평균으로 결합하여 과적합을 줄이고 일반화 성능을 높이는 앙상블 모델 |
| K-NN | 새로운 데이터와 가장 가까운 K개 이웃의 라벨을 기반으로 분류·예측하는, 학습 단계에서 파라미터를 학습하지 않는 메모리 기반 알고리즘 |
| SVM | 마진을 최대화하는 초평면을 찾아 데이터를 최적으로 분리하는 지도 학습 알고리즘 — 서포트 벡터만으로 결정 경계를 정의 |
| Naive Bayes | 모든 특성이 서로 독립이라고 가정하고 베이즈 정리로 각 클래스에 속할 확률을 계산하는 분류 모델 — 빠르고 텍스트 분류에 강함 |

## 딥러닝 기초

| 개념 | 한 줄 정리 |
| --- | --- |
| Deep Learning | 여러 층의 신경망을 통해 데이터의 비선형 패턴을 자동으로 학습하여 예측·분류하는 머신러닝의 하위 분야 |
| Perceptron | ANN의 기본 단위 — 입력값에 가중치를 곱한 가중합을 활성화 함수에 적용하여 단일 출력을 생성하는 선형 이진 분류기 |
| Activation Function | 뉴런의 가중합을 비선형 변환하여 신경망이 복잡한 패턴을 학습할 수 있게 하는 함수 (Sigmoid, Tanh, ReLU) |
| ANN | 뇌의 뉴런 구조를 모방한 알고리즘 — 입력층·은닉층·출력층을 거쳐 데이터의 패턴을 학습하고 예측을 수행 |
| FCNN | 각 층의 모든 뉴런이 다음 층의 모든 뉴런과 완전히 연결된 신경망 구조 — ANN의 한 종류 |
| FCL | ANN에서 모든 입력 뉴런이 모든 출력 뉴런과 연결된 계층 (TensorFlow: Dense Layer / PyTorch: Linear Layer) |
| Loss Function | 모델의 예측값과 실제값 간의 차이를 정량적으로 측정하는 함수 — 학습 과정에서 가중치 조정의 기준이 됨 |
| Backpropagation | 출력에서 발생한 손실을 체인 룰(Chain Rule)로 역방향 전파하여 각 가중치의 기울기(Gradient)를 계산하는 알고리즘 |
| Optimizer | 역전파로 계산된 기울기를 바탕으로 실제 가중치를 업데이트하여 손실을 최소화하는 최적화 알고리즘 |

## 사전 훈련 모델

| 개념 | 한 줄 정리 |
| --- | --- |
| Pre-trained Model | 대규모 데이터셋으로 미리 학습이 완료된 모델 — 저수준~고수준 특징을 이미 학습한 가중치로 시간·데이터·자원 절약 |
| VGG16 | 3×3 합성곱 필터를 균일하게 쌓은 16층 구조 — 단순하고 직관적이어서 특징 추출·교육 목적으로 많이 사용 |
| ResNet | Skip Connection(잔차 연결)으로 기울기 소실 없이 매우 깊은 네트워크 학습이 가능한 사전 훈련 모델 |

## 전이 학습

| 개념 | 한 줄 정리 |
| --- | --- |
| Transfer Learning | 사전 훈련된 모델의 가중치를 새로운 문제에 재활용하여 학습 시간·데이터·자원을 절약하는 기법 |
| Fine Tuning | 사전 훈련 모델의 일부 레이어를 해동하여 새 도메인 데이터로 재학습하는 과정 — Partial / Full / 단계적 / 하이브리드 방식으로 구분 |
| Feature Extraction | 사전 훈련 모델 전체를 고정하고 새 분류 헤드만 학습하는 방법 — 소량 데이터에서 빠른 학습 가능 |
| Zero-Shot Learning | 별도 학습 없이 사전 훈련 모델(CLIP 등)로 새로운 클래스를 추론하는 기법 |
| 성능 시각화 | 학습·검증 과정의 Accuracy / Loss 곡선을 matplotlib으로 그려 과적합·수렴 여부를 확인하는 방법 |

## 모델 튜닝

| 개념 | 한 줄 정리 |
| --- | --- |
| Overfitting (과적합) | 학습 데이터를 과하게 학습하여 새로운 데이터 오차가 증가하는 현상 — Dropout·정규화·Early Stopping으로 방지 |
| Underfitting (과소적합) | 모델이 학습 데이터의 패턴조차 제대로 학습하지 못하는 현상 — 모델 복잡도 향상·ReLU 활성화 함수·깊은 구조로 개선 |
| Early Stopping | 검증 손실이 개선되지 않으면 patience 횟수 대기 후 학습을 조기 종료하여 과적합을 방지하는 기법 |
| Hyperparameter Tuning | 학습률·배치 크기·레이어 수 등 학습 전 설정해야 하는 값을 Grid Search / Random Search로 최적화하는 과정 |

## NLP

| 개념 | 한 줄 정리 |
| --- | --- |
| NLP | 인간의 자연어와 컴퓨터를 연결해 텍스트 전처리 → 형태소 분석 → 벡터화 → 구문 분석 과정으로 자연어를 이해·처리하는 AI 분야 |
| RNN | 시퀀스 데이터를 처리하기 위해 hidden state로 이전 시점 정보를 기억하는 신경망 — 시퀀스가 길어지면 기울기 소실(Vanishing Gradient) 문제 발생 |
| LSTM | Forget·Input·Output 게이트와 셀 상태로 RNN의 기울기 소실 문제를 해결한 신경망 — 장/단기 기억을 통해 긴 시퀀스의 의존성 학습 가능 |
